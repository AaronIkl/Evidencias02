# Evidencias 02: Portafolio de Proyectos

Este repositorio contiene las actividades realizadas.

---

## 1. Actividad 5 - Menu Lateral
(Archivo: `Actividad5_MenuLateral.apk`)

* **Reseña:** Esta aplicación implementa un panel de navegación lateral que permite al usuario moverse entre las principales secciones de la app. 
* **Experiencia:** La implementación de este componente fue clave para la gestión de Fragments y la navegación entre pantallas. 

## 2. Actividad 6 - Menu Opciones App
(Archivo: `Actividad6_MenuOpciones.apk`)

* **Reseña:** Se desarrolló una app que utiliza el 'Options Menu'. Este menú se utiliza para alojar acciones importantes, como 'Ajustes', 'Buscar' o 'Compartir'.
* **Experiencia:** Definir los menús mediante recursos XML y a gestionar los eventos de clic. Fue una práctica útil para manejar la interacción del usuario con la barra de herramientas.

## 3. Actividad 7 - My Movies
(Archivo: `Actividad7_MyMovies.apk`)

* **Reseña:** La aplicación modela y muestra al usuario en una interfaz limpia.
* **Experiencia:** La experiencia principal fue elegir las películas.

## 4. Actividad 8 - Listados
(Archivo: `Actividad8_Listados.apk`)

* **Reseña:** Esta aplicación implementa un sistema CRUD (Crear, Leer, Editar, Eliminar) completo, como pedía la actividad. El tema elegido fue "Especies de Focas". La app permite al usuario agregar nuevas especies con varios campos.
* **Experiencia:** fue facilito, porque el profe enseña muy bien.
